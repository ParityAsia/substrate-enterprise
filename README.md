# Substrate 联盟链开发

## 目标

基于 Substrate 进行底层开发，完成 [《金融分布式账本技术安全规范》](https://www.cfstc.org/bzgk/gk/view/bzxq.jsp?i_id=1855) 规定的内容。

## 原则

* 构建长期支持、稳定性良好的 Substrate 联盟链功能特性；
* 开源，并尽可能贡献到主仓库。
